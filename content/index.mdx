## About Canopy

This is demo using Ira Aldridge photographs provided by Northwestern University Libraries. The original source for these photographs is from this filtered search result `https://dc.library.northwestern.edu/search?q=ira+aldridge&genre=photographs` at Northwestern University Libraries. These are provided through the Northwestern Digital Collections API and made available to the end-user through the View as IIIF dropdown menu.
